掌機 - Miyoo Mini Plus - Extract Device Tree


步驟如下： 

$ pip install extract-dtb

$ cd
$ wget https://github.com/steward-fu/website/releases/download/miyoo-mini-plus/mtdblock1
$ dd if=mtdblock1 of=kernel.xz bs=1 skip=64
$ 7za x kernel.xz
$ extract-dtb kernel
$ cp dtb/04_dtbdump_INFINITY2M_SSC011A-S01A-S.dtb input.dtb
$ dtc -I dtb -O dts -f input.dtb -o output.dts
