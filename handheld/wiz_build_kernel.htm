掌機 - GP2X Wiz - Build Kernel


步驟如下： 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/wiz/GPH_SDK-10.02_linux.tar.gz
$ tar xvf GPH_SDK-10.02_linux.tar.gz
$ sudo mv GPH_SDK /opt/
$ export PATH=/opt/GPH_SDK/tools/gcc-4.0.2-glibc-2.3.6/arm-linux/bin/:$PATH

$ wget https://github.com/steward-fu/website/releases/download/wiz/src_kernel.tar.gz
$ tar xvf src_kernel.tar.gz
$ cd kernel
$ ARCH=arm CROSS_COMPILE=arm-linux- make pollux_wiz_defconfig
$ ARCH=arm CROSS_COMPILE=arm-linux- make gk
