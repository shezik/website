Visual Memory Unit

介紹

司徒對於Dreamcast遊戲機的印象，僅止於它是Sega Saturn的後繼機種，只可惜這款機器存在一些問題，最終退出家用機市場，當初並沒有印象VMU(Visual Memory Unit)或者稱為VMS(Visual Memory System)這一款掌機的存在，不過這款掌機有一個特別的玩法，那就是除了可以結合Dreamcast遊戲主機或手把使用之外，也可以當作一台獨立掌機使用，更可以兩台掌機連線對打，而直到最近，司徒想要尋找一款小巧的掌機外殼，用來充當改機的材料，這才得知有這款掌機的存在，雖然年代久遠，不過目前還是可以找到一些極具參考的資料，加上目前可以用相當低廉的價格入手，因此，司徒也入手幾台玩玩，主要用途還是作為改機使用，雖然這款掌機麻雀雖小、五臟俱全且可以運行自製遊戲，不過由於效能以及屏幕有點差，因此，如果不是當作改機用途，會比較適合當作研究用途，不過關於這方面的資訊就顯得相當稀少，司徒之後有時間再好好整理一下。
