掌機 - Pandora(Rebirth) - SuperZaxxon - C/C++ - 開發環境


參考資訊：
https://www.scribblygum.com.au/blog/post/setting-up-a-c-development-environment-on-the-pandora/

步驟如下： 

$ sudo opkg install gcc gcc-symlinks gcc-doc make make-dev binutils-dev
$ sudo opkg install cpp cpp-symlinks
$ sudo opkg install g++ g++-symlinks libstdc++-dev
$ sudo opkg install libgles-omap3-es23-dev
$ sudo opkg install libsdl-1.2-dev libsdl-gfx-dev libsdl-image-1.2-dev libsdl-net-1.2-dev libsdl-ttf-2.0-dev
$ sudo opkg --nodeps install libsdl-mixer-1.2-dev
$ sudo opkg install git
