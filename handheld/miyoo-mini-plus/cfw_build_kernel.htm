build kernel

由於SigmaStar並沒有公開Video和Audio的相關程式碼，因此，司徒使用GitHub可以找到的Kernel進行測試，發現搭配Miyoo Mini Plsu的官方驅動程式是可以運作的，而Device Tree的部份則是從官方驅動提取出來使用，所以除了Video和Audio沒有程式碼之外，司徒正慢慢將其餘部份補齊，編譯的步驟如下： 

$ cd
$ git clone https://github.com/steward-fu/kernel
$ cd kernel
$ git checkout ssd202d_miyoo-mini-plus_linux-4.9.84
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- make mmp_defconfig
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- make all -j4

P.S. arch/arm/boot/uImage.xz
