掌機 - Pandora(Rebirth) - SuperZaxxon - Build PCSX ReARMed


步驟如下： 

$ cd
$ git clone https://github.com/notaz/c64_tools
$ sudo cp -a c64_tools/include* /usr/include/
$ sudo cp -a c64_tools/dsp /usr

$ cd
$ git clone https://github.com/steward-fu/pcsx_rearmed
$ cd pcsx_rearmed
$ git submodule update --init --recursive
$ ./configure --platform=pandora
$ make
$ make rel
