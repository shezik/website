掌機 - Pandora(Rebirth) - SuperZaxxon - 如何從SD開機


參考資訊：
https://pandorawiki.org/Boot_setup#OR_copy_your_firmware_from_the_NAND_to_the_SD_card

步驟如下：
1. 第1磁區格式化成ext3
2. autoboot.txt 

setenv bootargs root=/dev/mmcblk0p1 rw rootwait vram=6272K omapfb.vram=0:3000K mmc_core.removable=0
ext2load mmc 0 0x80300000 /boot/uImage
bootm 0x80300000

3. 複製Pandora根目錄的所有檔案到ext3磁區
4. SD插入左邊卡槽並且開機即可
