掌機 - Pandora (Rebirth) - SuperZaxxon - 如何透過USB Mini B登入SSH


參考資訊：
https://pyra-handheld.com/boards/threads/please-explain-usb-options.74048/

開啟步驟： 

$ sudo vim /etc/network/interfaces
    auto usb0
    iface usb0 inet static
        address 10.0.1.3
        netmask 255.255.255.0

$ sudo modprobe -r g_cdc
$ sudo modprobe g_ether

連接USB Mini B
