掌機 - Miyoo Mini - Parasyte - Build Buildroot


步驟如下： 

$ sudo chmod 0777 /opt

$ cd
$ wget https://github.com/steward-fu/website/releases/download/miyoo-mini/gcc-arm-8.2-2018.08-x86_64-arm-linux-gnueabihf.tar.gz
$ tar xvf gcc-arm-8.2-2018.08-x86_64-arm-linux-gnueabihf.tar.gz
$ sudo mv gcc-arm-8.2-2018.08-x86_64-arm-linux-gnueabihf /opt/prebuilt

$ wget https://github.com/steward-fu/website/releases/download/miyoo-mini/buildroot-2021.11.tar.gz
$ tar xvf buildroot-2021.11.tar.gz
$ cd buildroot-2021.11/

$ wget https://github.com/steward-fu/website/releases/download/miyoo-mini/buildroot-2021.11_config
$ mv buildroot-2021.11_config .config
$ make clean
$ make -j

P.S. toolchain=/opt/mmiyoo/
