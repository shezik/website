掌機 - Miyoo A30 - 解決Symbol __libc_readline_unlocked問題


問題如下： 

root@MIYOO282:/mnt/SDCARD/App/balatro# chroot mnt
chroot: relocation error: lib/libnss_files.so.2: symbol __libc_readline_unlocked, version GLIBC_PRIVATE not defined in file libc.so.6 with link time reference

解法如下(使用Target OS的libnss_files.so.s)： 

LD_PRELOAD=libnss_files.so.2 chroot mnt
