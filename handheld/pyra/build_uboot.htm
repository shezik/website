Pyra

build uboot

參考資訊：
1. pyra-alt-ddr3
2. debian-getting-started-with-the-omap5432-uevm

步驟如下： 

$ cd
$ git clone https://dev.pyra-handheld.com/daveshah/pyra-uboot/ -b pyra-alt-ddr3
$ cd pyra-uboot
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- make omap5_pyra_4gb_defconfig
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- make -j4

$ sudo dd if=/dev/zero of=/dev/sdX bs=1M count=10
$ sudo dd if=MLO of=/dev/sdX count=2 seek=1 bs=128k
$ sudo dd if=u-boot.img of=/dev/sdX count=4 seek=1 bs=384k
