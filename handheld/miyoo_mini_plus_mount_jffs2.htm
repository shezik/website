掌機 - Miyoo Mini Plus - 如何掛載JFFS2檔案


參考資訊：
http://blog.gitdns.org/2015/06/15/2015-06-15-mount-jffs2/

如下： 

$ sudo apt-get install mtd-tools -y

$ sudo modprobe jffs2
$ sudo modprobe mtdram
$ sudo modprobe mtdblock

$ sudo dd if=mtdblock7 of=/dev/mtd0
$ sudo mount -t jffs2 /dev/mtdblock0 /mnt
$ ls /mnt/
    LCM.ini  nsswitch.conf  romwinidx.json  system.json  wpa_supplicant.conf  wpa_supplicant.conf_bak
