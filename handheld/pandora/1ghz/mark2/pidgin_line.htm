如何在Pidgin上使用LINE聊天軟體

司徒目前使用的主要手機是N900，該手機使用Maemo系統並且沒有LINE聊天軟體的支援，加上在台灣主要使用的聊天軟體就是LINE軟體，雖然司徒不是很常使用聊天軟體，但是礙於家人都是使用LINE聊天軟體，因此，司徒必需尋找是否可以在Linux系統下使用LINE聊天軟體的方法，結果找到的方式都是使用Wine執行PC版的LINE聊天軟體，這種方式無法在司徒的N900和Pandora身上執行，因為這兩台機器都是ARM CPU；非Intel x86 CPU，雖然不意外搜尋結果，但是司徒還是不放棄，最後終於找到日本網站介紹如何編譯Pidgin使用的LINE插件，經過多次編譯測試，司徒最後終於成功編譯出N900(EasyDebian)和Pandora(Debian Mark2)可以使用的Pidgin插件，步驟如下說明：

安裝必要套件 

$ sudo apt-get update
$ sudo apt-get install pidgin libpurple0 libpurple-dev libpurple-bin libboost-dev libboost-test-dev libboost-program-options-dev libevent-dev automake libtool flex bison pkg-config g++ libssl-dev

編譯歩驟如下： 

$ git clone https://github.com/steward-fu/gh_pidgin.git
$ cd linei/purple-line
$ make thrift
$ make
$ sudo make install
$ sudo cp libline.so /usr/lib/purple-2/

完成

接著添加完LINE帳號後，Pidgin會跳出LINE的4碼認證號碼，只要在手機上確認後，就可以在N900和Pandora上使用LINE聊天軟體。
