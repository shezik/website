如何在Pidgin上使用Skype聊天軟體

雖然司徒已經編譯出可以在Pidgin上使用的LINE聊天軟體插件，但是司徒還是想儘可能的使用更多聊天軟體，於是司徒鎖定Skype聊天軟體，上網找了一下後，發現如下方式可以編譯出Pidgin使用的Skype插件。

安裝必要套件 

$ sudo apt-get update
$ sudo apt-get install pidgin libpurple0 libpurple-dev libpurple-bin libboost-dev libboost-test-dev libboost-program-options-dev libevent-dev automake libtool flex bison pkg-config g++ libssl-dev

編譯歩驟如下： 

$ git clone https://github.com/steward-fu/gh_pidgin.git
$ cd skype/skypeweb
$ make
$ sudo make install
$ sudo cp libskypeweb.so /usr/lib/purple-2/
