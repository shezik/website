安裝系統

Pandora掌機可以選擇安裝的作業系統真的相當多，除了Slackware、Debain、Android系統之外，包含目前相當熱門的Ubuntu系統也是可以選擇的系統之一，如果使用者喜愛Ubuntu系統的操作介面，司徒建議安裝嚐鮮看看，不過對於PND的安裝支援，司徒覺得除了原廠系統之外，Slackware系統還是支援Pandora掌機最完整的系統，不過Slackware解壓縮後的安裝檔案真的相當大，至少需要8GB SDCard才可以安裝，反觀Ubuntu系統，它的安裝檔案也不過才510MB大小(解壓縮後是1.5GB)，所以準備一張4GB SDCard即可安裝，安裝步驟也是相當簡單，值得嚐鮮看看，接著司徒將安裝過程說明如下。

安裝步驟：
1. 下載Ubuntu 12.04
2. 準備一張4GB SDCard並製作2個分割表：512MB(swap)、3.25GB(ext2)(順序不能變更)，標籤名稱請使用ubuntu
3. 執行如下命令： 

$ sudo tar xvf ubuntu-12.04-preinstalled-desktop-armhf+omap-pandora1.tar.gz -C /media/ubuntu

4. 由於需要SuperZaxxon系統的相關檔案，因此，進入SuperZaxxon系統後，將SDCard插入左邊插槽並執行如下命令： 

$ sudo mkdir -p /media/ubuntu/boot
$ sudo cp /boot/uImage-3 /media/ubuntu/boot/
$ sudo cp -rf /lib/modules/3.2.21/* /media/ubuntu/lib/modules/

P.S. module後面的版號會隨系統版本而改變。

5. 更改/media/ubuntu/boot.txt的/lib/boot/uImage成/boot/uImage-3
6. 重新開機並按住R按鈕，出現開機選單後，選擇boot from SD1:2並按B按鈕即可進入Ubuntu系統(帳號：pandora、密碼：p)
