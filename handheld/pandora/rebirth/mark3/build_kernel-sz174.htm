Pandora Rebirth >> Debian Mark3

build kernel 3.2.78

編譯方式如下： 

$ cd 
$ git clone git://git.openpandora.org/pandora-kernel.git
$ cd pandora-kernel
$ git checkout tags/sz_174
$ make omap3_pandora_defconfig
$ make uImage modules
$ make modules_install INSTALL_MOD_PATH=out

編譯完成後，把arch/arm/boot/uImage覆蓋掉原本/boot/uImage-3，然後把out/lib/底下的檔案複製到/lib/下，重新開機後執行depmod -a即可。
