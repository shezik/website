掌機 - Pandora(Rebirth) - SuperZaxxon - 安裝開發環境


參考資訊：
https://www.scribblygum.com.au/blog/post/setting-up-a-c-development-environment-on-the-pandora/

安裝方式： 

$ sudo opkg update
$ sudo opkg install gcc gcc-symlinks gcc-doc make make-dev binutils-dev cpp cpp-symlinks vim unzip g++ g++-symlinks libstdc++-dev libgles-omap3-es23-dev libgles-omap3-dm3730-dev libsdl-1.2-dev libsdl-gfx-dev libsdl-image-1.2-dev libsdl-net-1.2-dev libsdl-ttf-2.0-dev
$ sudo opkg --nodeps install libsdl-mixer-1.2-dev
