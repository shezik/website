掌機 - Pandora (Rebirth) - Debian (Mark3) - 如何啟動/dev/ttyO2


參考資訊：
https://www.rogerirwin.co.nz/linux-open-source/enabling-a-serial-port-console

步驟如下： 

$ sudo vim /lib/systemd/system/serial-getty@.service
    ExecStart=-/sbin/agetty 115200 %I $TERM

$ sudo systemctl daemon-reload
$ sudo systemctl enable serial-getty@ttyO2.service
$ sudo systemctl start serial-getty@ttyO2.service
