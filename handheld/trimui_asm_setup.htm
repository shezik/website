掌機 - TRIMUI - Assembly - 開發環境


司徒使用GNU GCC編譯環境，安裝方式如下： 

$ sudo apt-get install gcc-arm-none-eabi-*

燒錄軟體則是sunxi-tools 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/trimui/sunxi-tools.7z
$ 7zr x sunxi-tools.7z
$ cd sunxi-tools
$ make clean
$ make

由於Allwinner韌體有Checksum檢查欄位，因此，需要編譯mksunxi工具，該工具可以針對韌體計算更新Checksum欄位 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/trimui/mksunxi.c
$ gcc mksunxi.c -o mksunxi
