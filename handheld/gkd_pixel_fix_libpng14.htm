掌機 - GKD Pixel - 解決skipping incompatible libpng14.so when searching for -lpng14問題


問題如下： 

/usr/bin/ld: skipping incompatible /opt/gcw0-toolchain/usr/mipsel-gcw0-linux-uclibc/sysroot/usr/lib/libpng14.so when searching for -lpng14
/usr/bin/ld: skipping incompatible /opt/gcw0-toolchain/usr/mipsel-gcw0-linux-uclibc/sysroot/usr/lib/libpng14.a when searching for -lpng14
/usr/bin/ld: cannot find -lpng14: No such file or directory
/usr/bin/ld: skipping incompatible /opt/gcw0-toolchain/usr/mipsel-gcw0-linux-uclibc/sysroot/usr/lib/libpng14.so when searching for -lpng14
collect2: error: ld returned 1 exit status
make[1]: *** [scripts/Makefile.host:118: scripts/png2logo] Error 1
make: *** [Makefile:518: scripts] Error 2

解法如下： 

$ vim scripts/Makefile
    HOSTLOADLIBES_png2logo = `/usr/bin/pkg-config --libs libpng`
