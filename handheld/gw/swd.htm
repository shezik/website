連接ST-LINK V2

P.S. VDD接到3.3V

編譯openocd 

$ cd
$ git clone git://repo.or.cz/openocd.git
$ cd openocd
$ ./bootstrap
$ ./configure --prefix=/usr
$ make
$ sudo make install

連接 

$ sudo openocd -f /usr/local/share/openocd/scripts/interface/stlink.cfg -f /usr/local/share/openocd/scripts/target/stm32h7x.cfg
