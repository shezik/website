build xboot

參考資訊：
1. get_started

編譯步驟如下： 

$ cd
$ wget https://github.com/steward-fu/miyoo/releases/download/v1.0/toolchain.7z
$ 7za x toolchain.7z
$ sudo cp -a miyoo /opt/
$ export PATH=$PATH:/opt/miyoo/bin

$ git clone -b f1c100s-kk131 https://github.com/xboot/xboot.git
$ cd xboot
$ make CROSS_COMPILE=arm-linux- PLATFORM=arm32-f1c500s

開機測試 

$ wget https://github.com/steward-fu/lichee-nano/releases/download/v1.0/f1c100s_ddr_init.bin

$ sudo sunxi-fel spl f1c100s_ddr_init.bin
$ sudo sunxi-fel -p write 0x80000000 output/xboot.bin
$ sudo sunxi-fel exec 0x80000000
