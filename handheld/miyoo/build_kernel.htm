Miyoo

build kernel 4.14.0

編譯步驟如下： 

$ cd
$ wget https://github.com/steward-fu/miyoo/releases/download/v1.2/kernel.7z
$ 7za x kernel.7z
$ cd kernel
$ make ARCH=arm CROSS_COMPILE=arm-linux- miyoo_defconfig
$ make ARCH=arm CROSS_COMPILE=arm-linux- clean
$ make ARCH=arm CROSS_COMPILE=arm-linux- -j8

燒錄到SPI Flash 

sudo sunxi-fel -p spiflash-write 1048576 arch/arm/boot/dts/suniv-f1c500s-miyoo.dtb
sudo sunxi-fel -p spiflash-write 1114112 arch/arm/boot/zImage
