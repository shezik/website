掌機 - Pyra - Build Kernel


參考資訊：
https://projects.goldelico.com/p/gta04-kernel
https://dev.pyra-handheld.com/daveshah/pyra-uboot/-/tree/pyra-alt-ddr3
https://forum.digikey.com/t/debian-getting-started-with-the-omap5432-uevm/12455

步驟如下： 

$ cd
$ git clone http://git.goldelico.com/letux-kernel.git
$ cd letux-kernel
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- make pyra_defconfig
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- make all -j4

P.S. zImage、omap5-letux-cortex15-v5.3+pyra-v5.3.dtb
