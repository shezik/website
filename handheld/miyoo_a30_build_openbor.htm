掌機 - Miyoo A30 - Build OpenBOR


參考資訊：
https://github.com/DCurrent/openbor
https://github.com/steward-fu/website/releases/download/miyoo-a30/a30_toolchain.tar.gz

步驟如下： 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/miyoo-a30/a30_toolchain.tar.gz
$ tar xvf a30_toolchain.tar.gz
$ sudo mv a30 /opt

$ wget https://github.com/steward-fu/website/releases/download/miyoo-a30/src_emu_openbor-v1.0.zip
$ unzip src_emu_openbor-v1.0.zip
$ src_openbor/engine/
$ make BUILD_A30=1 clean
$ make BUILD_A30=1 -j4
