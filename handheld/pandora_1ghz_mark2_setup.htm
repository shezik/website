安裝步驟：
1. 執行如下命令： 

$ cd
$ wget http://pandian.openpandora.org/releases/pandian-mark2-hf_2014-01-02.7z
$ 7za x pandian-mark2-hf_2014-01-02.7z
$ sudo dd if=pandian-mark2-hf_2014-01-02.img of=/dev/sdX bs=1M

2. 將第2分區調整到最大空間
3. 開機並且進入Debian系統後

5. ~/.xbindkeysrc 

"gnome-screenshot -w" 
    control + alt + w

"gnome-screenshot" 
    control + alt + s

"lxpanelctl menu"
    XF86MenuKB

按鍵說明
Ctrl + Alt + wScreen catpture(當下視窗)
Ctrl + Alt + sScreen catpture(全螢幕)
PandoraMenu

6. 修改Repository 

$ sudo nano /etc/apt/sources.list.d/pandora.list
    deb [arch=armhf] http://ftp.us.debian.org/debian jessie main contrib non-free
    deb [arch=armhf] http://ftp.us.debian.org/debian wheezy main contrib non-free
    deb-src [arch=armhf] http://ftp.us.debian.org/debian jessie main contrib non-free
    deb-src [arch=armhf] http://ftp.us.debian.org/debian wheezy main contrib non-free

7. 安裝套件 

$ sudo apt-get update
$ sudo apt-get install xbindkeys xmodmap

8. 更新按鍵和開啟聲音 

$ sudo amixer set Master,0 100% unmute
$ sudo mpd /etc/pandian/mpd.conf --kill
$ sudo mpd /etc/pandian/mpd.conf

$ setxkbmap en_US
$ xmodmap ~/.pndXmodmap
$ xbindkeys -f ~/.xbindkeysrc

9. 修改成自動登入 

$ sudo nano  /etc/lightdm/lightdm.conf
    autologin-user=xxx
    autologin-user-timeout=0

P.S. xxx是使用者名字

10. 修改螢幕亮度 

$ sudo pdcmd LCDBright 5

完成
