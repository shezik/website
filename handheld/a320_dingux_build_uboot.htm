掌機 - Dingoo A320 - Dingux - Build U-Boot


步驟如下： 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/a320/cc-dx.tar.gz
$ wget https://github.com/steward-fu/website/releases/download/a320/src_uboot.tar.gz

$ tar xvf cc-dx.tar.gz
$ sudo mv a320 /opt
$ export PATH=/opt/a320/usr/bin/:$PATH

$ tar xvf src_uboot.tar.gz
$ cd uboot
$ make distclean
$ make a320_config
$ make
