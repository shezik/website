停止Suspend(闔上螢幕時)

由於GPD Win掌機在Win10系統下會有休眠以及睡眠的問題，常常發生睡死的狀況，因此，司徒為了避免這樣的悲劇發生在Debian身上，只好預設就把Suspend關閉，過程如下說明。

首先安裝Power相關套件 

$ sudo apt-get install xfce4-power-manager
$ sudo xfce4-power-manager-settings

接著設定闔上上蓋只要關閉螢幕背光就好


停止系統閒置設定功能


停止螢幕閒置設定功能


關閉螢幕保護程式鎖住功能


完成

P.S. 闔上螢幕後，風扇也會跟停止轉動。
