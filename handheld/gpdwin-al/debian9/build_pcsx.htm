build pcsx rearmed

不知從何開始，如果掌機可以跑順PS1模擬器，基本上就具備神機的入選資格，久而久之，當要幫掌機移植模擬器時，在司徒心中，PS1模擬器就會變成是第一首選，因此，當拿到GPD Win掌機時，司徒第一時間就是想在GPD Win上跑PS1模擬器，所幸在Debian系統下，可以很輕易的編譯PCSX ReARMed模擬器，過程如下說明： 

$ cd ~/Downloads/
$ git clone https://github.com/notaz/pcsx_rearmed.git
$ cd pcsx_rearmed/frontend
$ git submodule update --init --recursive
$ cd ../
$ ./configure --platform=maemo

雖然已經可以執行PS1模擬器，不過目前只能使用SDL介面，因此，解析度只能使用PS1游戲的Native Resolution，像是超級機器人大戰F就只能使用320x240的解析度，如要可以全螢幕，則需研究一下程式，看看為何無法使用其它介面：如Overlay
