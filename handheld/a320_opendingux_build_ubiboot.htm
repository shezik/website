掌機 - Dingoo A320 - OpenDingux - Build UBIBoot


步驟如下： 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/a320/cc-dx.tar.gz
$ wget https://github.com/steward-fu/website/releases/download/a320/src_ubiboot.tar.gz

$ tar xvf cc-dx.tar.gz
$ sudo mv a320 /opt
$ export PATH=/opt/a320/usr/bin/:$PATH

$ tar xvf src_ubiboot.tar.gz
$ cd ubiboot
$ make CONFIG=a320 clean
$ make CONFIG=a320
