如何在UBoot添加SPI Flash型號

drivers/mtd/spi/spi_flash_ids.c 

{"25Q064A", INFO(0x20ba17, 0x0, 64 * 1024, 128, SECT_4K)},

arch/arm/dts/suniv-f1c100s-licheepi-nano.dts 

&spi0 {
  pinctrl-names = "default";
  pinctrl-0 = <&spi0_pins_a>;
  status = "okay";

  flash@0 {
    #address-cells = <1>;
    #size-cells = <1>;
    compatible = "winbond,25q064a", "jedec,spi-nor";
    reg = <0>;
    spi-max-frequency = <40000000>;
  };
};

執行後 

SF: Detected 25Q064A with page size 256 Bytes, erase size 4 KiB, total 8 MiB
