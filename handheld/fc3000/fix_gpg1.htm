解決"Release signed by unknown key (key id 6FB2A1C265FFB764)"問題

參考資訊：
1. debootstrap-release-signed-by-unknown-key

問題： 

$ sudo debootstrap --arch armel --foreign wheezy wheezy http://archive.debian.org/debian/
    I: Retrieving InRelease
    I: Retrieving Release
    I: Retrieving Release.gpg
    I: Checking Release signature
    E: Release signed by unknown key (key id 6FB2A1C265FFB764)
       The specified keyring /usr/share/keyrings/debian-archive-keyring.gpg may be incorrect or out of date.
       You can find the latest Debian release key at https://ftp-master.debian.org/keys.html

解法： 

$ wget https://ftp-master.debian.org/keys/release-7.asc -qO- | gpg --import --no-default-keyring --keyring ./debian-release-7.gpg
$ sudo debootstrap --keyring=./debian-release-7.gpg --arch armel --foreign wheezy wheezy http://archive.debian.org/debian/
