掌機 - Pandora (1GHz) - SuperZaxxon - 中文輸入法


更新套件 

$ sudo opkg update
$ sudo opkg install gtk-immodule-xim
$ sudo opkg install angstrom-locale-zh-feed-config
$ sudo opkg install locale-base-zh-tw

設定中文語系 

$ sudo vim ~/.profile
    export LANG=zh_TW.UTF-8
    export LC_ALL=zh_TW.UTF-8
    export XMODIFIERS=@im=uim
    export GTK_IM_MODULE=uim
    uim-xim &
    uim-toolbar-gtk &

支援中文檔名 

$ sudo nano /etc/udev/scripts/mount.sh
    if ! $MOUNT -t auto -o dirsync,noatime,umask=0,utf8 $DEVNAME "/media/$name" && ! $MOUNT -t auto -o dirsync,noatime,utf8 $DEVNAME "/media/$name"

設定vfat mount格式為utf8 

$ sudo nano /etc/xdg/xfce4/mount.rc
    [vfat]
    uid=
    utf8=true
    shotname=winnt

安裝套件 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/pandora/cht_pinyin.tar.gz
$ cd /
$ sudo tar xvfp ~/cht_pinyin.tar.gz

重新開機 

$ sudo reboot

完成

P.S. 點擊右下角視窗切換輸入法，Shift + Space則是On、Off的切換
