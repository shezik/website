Nintendo 3DS

改造電池、充電接頭

3DS厚電


變厚且不美觀


司徒已經用不到這個卡槽，因此，這個位置相當適合用來擺放電池


解焊後才發現CPU在這裡，把電池擺放在CPU上方，其實不是很好的改法


司徒將原本充電接頭改造成USB Type-C，因此，需要挖一個比較大的洞


電池使用503040 600mA、充電接頭改成USB Type-C

P.S. 電池+要接到原本的電池座，這張圖後來忘記更新～

OpenSCAD

$fn = 50;

difference() {
    cube([14.5, 4, 4.5]);
    translate([4.5, 5, 1.5]) {
        rotate([90, 0, 0]) {
            hull() {
                cylinder(5, 1.7, 1.7);
                translate([6, 0, 0]) {
                    cylinder(5, 1.7, 1.7);
                }
            }
        }
    }
}

相當前衛的顏色，卡槽之後可以在買一個防塵塞來使用


剛剛好沒有凸出
