掌機 - Pandora (1GHz) - Debian (Mark2) - 掛載主板上的Flash


Mount Flash 

$ sudo ubiattach -m 4 -d 0
$ sudo mount -t ubifs /dev/ubi0_0 /mnt
$ sudo ubiattach -m 3 -d 1
$ sudo mount -t ubifs /dev/ubi1_0 /mnt/boot

Umount Flash 

$ sudo umount /mnt/boot
$ sudo umount /mnt
$ sudo ubidetach -m 3
$ sudo ubidetach -m 4
