GPD Win2 >> Debian 9.0

build ppsspp

基本上，在GPD Win掌機就可以流暢把玩PPSSPP模擬器，更別說在GPD Win2掌機身上，而GPD Win2掌機配備6寸高清螢幕(GPD Win掌機是5.5寸)，因此，玩PPSSPP模擬器的視覺享受又再度被深深感動了一次，編譯過程如下： 

$ cd
$ sudo apt-get install cmake libsdl2-dev
$ git clone https://github.com/hrydgard/ppsspp
$ cd ppsspp
$ git submodule update --init --recursive
$ ./b.sh
$ ./build/PPSSPPSDL

完成


如要支援Vulkan，請安裝如下套件： 

$ sudo apt-get install mesa-vulkan-drivers
