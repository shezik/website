安裝系統(LXDE)

司徒相當喜愛LXDE的輕量，因為隨著時間改變，硬體規格越來越強大，但是軟體UI耗用的CPU資源卻是越來越吃重，CPU如果把資源耗在華麗的GUI畫面上，司徒總是覺得相當可惜，因為對於司徒來說，把資源拿來跑更複雜的程式或許意義更重大，因此，司徒的機器通常偏愛CPU使用率較低的桌面環境，如：LXDE或者XFCE4，所以司徒的GPD Win2掌機一到手後，司徒就馬上安裝Debian系統。

1. 下載https://cdimage.debian.org/debian-cd/current/amd64/iso-dvd/debian-9.5.0-amd64-DVD-1.iso
2. 製作USB開機碟
3. 開機按Delete鍵進入BIOS模式並且設定USB開機


修改後，儲存離開


選擇GUI安裝介面


英語鍵盤


接著預設安裝就可以完成(記得在選擇套件時，選擇LXDE)


HDMI輸出

MicroSD支援128GB


Fix Wifi 

$ sudo apt-get remove wicd*
$ sudo apt-get install firmware-iwlwifi
$ sudo apt-get install network-manager-gnome

Fix Screen 

$ xrandr -o right



Fix Backlight 

$ sudo chmod 0777 /sys/class/backlight/intel_backlight/brightness
$ echo 2000 > /sys/class/backlight/intel_backlight/brightness

Fix Gamepad 

$ sudo modprobe xpad
$ sudo chmod 0777 /sys/bus/usb/drivers/xpad/new_id
$ echo 0x0079 0x18d4 > /sys/bus/usb/drivers/xpad/new_id
