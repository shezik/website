安裝系統(XFCE)

雖然司徒相當喜愛LXDE介面，因為佔用的RAM還是比XFCE來得少，不過對於GPD Win2掌機的快捷鍵以及螢幕DPI設定上，XFCE的支援度還是比LXDE來得好，而對於GPD Win2掌機的硬體效能來說，佔用RAM的微小差異，基本上可以忽略，因此，司徒還是建議使用者安裝XFCE，安裝過程如下說明：
1. 下載Debian 9.0 DVD
2. 設定USB開機(開機按下Delete進入BIOS)


3. 開始安裝



4. 選擇XFCE


5. 完成


6. 設定螢幕旋轉方向


7. 設定DPI


8. 其它設定 

$ cd
$ wget https://github.com/steward-fu/gpdwin2/releases/download/v1.0/util.7z
$ 7za x util.7z
$ cd gpdwin2_util
$ sudo ./run.sh
$ sudo reboot

完成
