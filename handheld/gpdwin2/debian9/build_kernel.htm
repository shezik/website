GPD Win2 >> Debian 9.0

build kernel 4.18.0

目前Kernel 4.18.0已經修復相當多的問題了，就連XPad切換回滑鼠模式的延遲也得以改善，當然也包含顯示的閃屏問題，而Debian 9.0預設的Kernel版本是4.9.0，預設的版本還有很多東西沒有修正，因此，建議使用者手動升級Kernel到4.18.0版本。 

$ cd
$ sudo apt-get install kernel-package
$ wget https://github.com/torvalds/linux/archive/v4.18.tar.gz
$ tar xvf v4.18.tar.gz
$ cd linux-4.18
$ make-kpkg clean
$ fakeroot make-kpkg --initrd --revision=0.0.1 kernel_image kernel_headers
