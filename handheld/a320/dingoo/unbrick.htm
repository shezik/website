Unbrick Tool

當A320掌機需要更新Firmware時，必須透過Unbrick Tool進行燒錄的動作，如果使用者有更換螢幕時，當然也是需要使用此工具更新Firmware。

操作步驟：
1. 下載UnbrickerTool檔案
2. 將A320掌機透過USB連接到電腦
3. 按住B按鈕 + Reset按鈕後，放開Reset按鈕
4. 出現安裝驅動程式畫面時，請使用剛剛下載的資料夾安裝驅動程式，安裝完後必須是如下裝置


5. 接著執行unbrick.exe並且選擇DL檔案、韌體檔案
6. 接著按Burning System按鈕並等待燒錄完成


如果使用者安裝過Dingux驅動程式時，會發現裝置管理員不是顯示Chinachip USB Loader，此時可以強制替換成Chainchip驅動程式，步驟如下：
1. 在裝置上按滑鼠右鍵
2. 選擇Update Driver..
3. 選擇No, not this time
4. 選擇Install from a list or specific location (Advanced)
5. 選擇Don't search, I will choose the driver to install
6. 選擇Chinachip USB Loader class
