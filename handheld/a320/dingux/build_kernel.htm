Dingoo A320 >> Dingux

build kernel

A320掌機興起的開源風潮就是因為Dingux系統的移植，讓這一台掌機更具開發研究的樂趣，而Dingux系統也移植到許多掌機的身上，如：NeoGeoX、A330、A380等掌機，因此有必要對Dingux系統做深入研究，將來自己也可以動手做移植的相關事情，在開始研究之前，最基本的步驟就是要自己動手編譯一次，這樣才可以開始研究之路，因此，司徒今天就先介紹一下如何編譯Dingux Kernel檔案。

編譯步驟如下： 

$ cd
$ wget https://github.com/steward-fu/a320/releases/download/v1.0/cc-dx.tar.gz
$ tar xvf cc-dx.tar.gz
$ sudo cp -a a320 /opt/a320
$ export PATH=$PATH:/opt/a320/usr/bin/

$ wget https://github.com/steward-fu/a320/releases/download/v1.1/dingux.7z
$ tar xvf kernel.tar.gz
$ cd kernel
$ make a320_defconfig
$ make zImage
