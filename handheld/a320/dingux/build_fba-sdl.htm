build fba-sdl

FBA模擬器可以說是相當熱門的大型街機模擬器，它可以支援的街機硬體種類相當豐盛，因此也被大量移植到許多遊戲平台身上，作為指標性的街機模擬器，司徒當然也要說明一下它的編譯過程，將來若有機會移植到其它遊戲平台身上時，成功的編譯方式就是移植的第一個步驟，因此，司徒今天就介紹一下如何編譯FBA模擬器。

使用如下方式編譯： 

$ cd
$ wget https://github.com/steward-fu/a320/releases/download/v1.0/cc-dx.tar.gz
$ tar xvf cc-dx.tar.gz
$ sudo cp -a a320 /opt/a320
$ export PATH=$PATH:/opt/a320/usr/bin/

$ wget https://github.com/steward-fu/a320/releases/download/v1.1/fba-sdl.tar.gz
$ tar xvf fba-sdl.tar.gz
$ cd fba-sdl
$ make -f Makefile.dingux-gcc clean
$ make -f Makefile.dingux-gcc
