掌機 - Pandora(1GHz) - Debian(Mark2) - Build Kernel


步驟如下： 

$ cd
$ git clone git://git.openpandora.org/pandora-kernel.git
$ cd pandora-kernel
$ make omap3_pandora_defconfig
$ make uImage modules
$ sudo INSTALL_MOD_PATH="/lib/modules/" make modules_install
$ sudo cp arch/arm/boot/uImage /boot/uImage-3.2.78-hf
$ sudo vim /boot/autoboot.txt
    setenv bootargs root=/dev/mmcblk0p2 rw rootwait console=ttyS0,115200n8 vram=6272K omapfb.vram=0:3000K mmc_core.removable=0 video=/dev/fb0
    fatload mmc 0 ${loadaddr} uImage-3.2.78-hf
    bootm ${loadaddr}

$ sudo reboot

完成
