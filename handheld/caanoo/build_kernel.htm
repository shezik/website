GP2X Caanoo

Build Kernel

編譯步驟如下： 

$ cd
$ wget https://github.com/steward-fu/caanoo/releases/download/v1.0/GPH_SDK-10.08-Linux.7z
$ 7za x GPH_SDK-10.08-Linux.7z
$ sudo cp -a GPH_SDK /opt/caanoo/
$ export PATH=$PATH:/opt/caanoo/tools/gcc-4.2.4-glibc-2.7-eabi/bin

$ wget https://github.com/steward-fu/caanoo/releases/download/v1.1/kernel.tar.gz
$ tar xvf kernel.tar.gz
$ cd kernel
$ make caanoo_update_defconfig
$ make zImage
