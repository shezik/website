掌機 - Pandora(1GHz) - Ubuntu - 安裝系統


安裝步驟：
1. 準備一張4GB SD並製作2個分割表：512MB(swap)、3.25GB(ext2)(順序不能變更)，標籤名稱使用ubuntu
2. 執行如下命令： 

$ cd
$ wget http://lxtek.de/pandora/ubuntu/ubuntu-12.04-preinstalled-desktop-armhf+omap-pandora1.tar.gz
$ sudo tar xvf ubuntu-12.04-preinstalled-desktop-armhf+omap-pandora1.tar.gz -C /media/ubuntu

3. 複製SuperZaxxon系統的驅動程式 

$ sudo mkdir -p /media/ubuntu/boot
$ sudo cp /boot/uImage-3 /media/ubuntu/boot/
$ sudo cp -a /lib/modules/3.2.21/* /media/ubuntu/lib/modules/

4. 更改/media/ubuntu/boot.txt的/lib/boot/uImage成/boot/uImage-3
5. 重新開機並按住R按鈕，出現開機選單後，選擇boot from SD1:2並按B按鈕即可進入Ubuntu系統(pandora:p)
