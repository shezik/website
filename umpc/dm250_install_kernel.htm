參考資訊：

步驟如下：
1. https://github.com/steward-fu/website/releases/download/dm250/DM200_DM250_kernel_installer_0.2.zip
2. 格式化SD卡成FAT32
3. 將DM200_DM250_kernel_installer_0.2/裡面的檔案複製到SD卡根目錄
4. 按下電源按鍵 + 左Alt + 右Shift
