Flash Image

參考資訊：
1. C.h.i.p.FlashCollection

步驟如下：
1. 執行如下命令 

$ sudo apt-get install android-tools-fastboot sunxi-tools u-boot-tools -y

$ cd
$ wget https://archive.org/download/C.h.i.p.FlashCollection/CHIP-tools.zip
$ wget https://github.com/steward-fu/pocketchip/releases/download/v1.0/stable-pocketchip-b126.7z

$ unzip CHIP-tools.zip
$ 7za x stable-pocketchip-b126

$ cd CHIP-tools

2. 連接FEL到GND
3. 開啟Pocket C.H.I.P.電源
4. 執行如下命令 

$ lsusb
