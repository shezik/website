Sharp IS01

安裝Android 2.2(Flash)

步驟如下：
1. 首先root該機器(請參考司徒的教學文章)
2. 下載IS01機器的驅動程式
3. 下載刷機檔案並解壓縮到C:\根目錄
4. 安裝Android SDK並將IS01機器插入電腦(記得安裝驅動程式)
5. 開啟root權限(透過IS01root_002)
6. 使用如下命令製作資料夾和將檔案放入Flash Memory 

C:\cm4is01> adb shell "busybox mkdir -p /data/cm4is01/data"
C:\cm4is01> adb shell "busybox mkdir -p /data/cm4is01/cache"
C:\cm4is01> adb push system.img /data/cm4is01/
C:\cm4is01> adb push cminstall.sh /data/cm4is01/
C:\cm4is01> adb push cmboot.img /data/cm4is01/
C:\cm4is01> adb shell
$ su
# busybox chmod 0755 /data/cm4is01/cminstall.sh
# sh /data/cm4is01/cminstall.sh

7. 重新開機後，輸入使用如下命令後即可開始使用 

# fix_permissions
# reboot recovery

P.S. 記得使用Android超頻工具設定CPU最高頻率，這樣機器就可以使用最佳速度執行，另外，由於目前的這種刷法是雙系統並存，因此原廠系統也會被保存下來，所以重新開機後，必須使用reboot recovery命令(於Terminal命令下)切換成Android 2.2系統
