Install OpenWRT(NAND)(reflash.sh)

Software USBBoot
1. Plugin USB Cable into PC
2. U + Reset(直到燒錄完成)

$ sudo apt-get install libconfuse0
$ sudo usbboot -c "boot;nerase 0 4096 0 0"
$ sudo usbboot -c "boot;nprog 0 openwrt-xburst-u-boot.bin 0 0 -n"
$ sudo usbboot -c "boot;nprog 1024 openwrt-xburst-uImage 0 0 -n"
$ sudo usbboot -c "boot;nprog 2048 openwrt-xburst-qi_lb60-boot.ubi 0 0 -n"
