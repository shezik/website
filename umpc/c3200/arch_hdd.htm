Zaurus SL-C3200 >> Arch Linux

如何安裝系統到HDD

參考資訊：
1. oesf
2. ZALARM-install

步驟：
1. 先將Arch Linux安裝到SDCard，透過SDCard開機進入系統
2. 命令如下 

# parted -s -a optimal /dev/sda mklabel msdos -- mkpart primary ext4 1 -1
# mkfs.ext4 /dev/sda1
# mount /dev/sda1 /mnt
# cd /mnt
# tar xvf xxx/alarm-zaurus-c3x00-minimal-rootfs-may2017.tar
# vim boot/boot.cfg
    LABEL=Arch Linux
    KERNEL=/boot/zImage-3.11.3-c3x00
    APPEND=root=/dev/sda1 rootfstype=ext4 fbcon=rotate:1 noinitrd

# cd
# umount /mnt

P.S. 透過開機選單進入系統
