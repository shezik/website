1. brainux

Brainux是一套可以跑在Sharp Brain翻譯機上的Linux系統，目前支援蠻多Sharp Brain機種，啟動Linux系統的方式，在比較舊的機種上，如：Sharp Brain PW-A7300，只能進入官方系統(Windows Embedded CE)後，再去啟動Linux系統，而製作Linux系統的方式也相當簡單，只要將燒錄檔案Clone到MicroSD就可以，步驟如下： 

$ cd
$ wget https://github.com/brain-hackers/buildbrain/releases/download/ci-timesyncd-tag/sdimage-ci-timesyncd.zip
$ unzip sdimage-ci-timesyncd.zip
$ sudo dd if=sdimage-ci-timesyncd.img of=/dev/sdX bs=1M

預設可以使用brain帳號登入，但是，官方系統是日文系統，司徒目前還不知道哪邊可以設定密碼，於是，只好修改MicroSD rootfs密碼比較快一點，步驟如下： 

$ sudo chroot /media/sdX

root@debian:/# passwd root
    New password:

    passwd: password updated successfully

root@debian:/# passwd user
    New password:
