Nokia N810

備份系統

1. 下載tools
2. 使用如下指令系統備份 

$ sudo apt-get update
$ sudo apt-get install wget e2fsprogs python2.5-runtime
$ sudo tar zxvf n810_bin_debpackages.tar.gz
$ cd debpackages
$ sudo dpkg -i console-tools.deb
$ sudo cp 0xFFFF.static /usr/bin
$ sudo chmod 0777 /usr/bin/0xFFFF.static
$ sudo mkdir -p /media/mmc2/backupSystem
$ sudo ./0xFFFF.static -e /media/mmc2/backupSystem/
$ ls /media/mmc2/backupSystem
  config.bin 
  initfs.jffs2 
  rootfs.jffs2
  secondary.bin 
  xloader.bin
  zImage
