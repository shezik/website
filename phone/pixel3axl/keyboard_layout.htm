司徒目前使用queekboard-terminal-layout作為Terminal的Keyboard Layout，然後，新增一個Custom Layout作為觸控遊戲手把使用(~/.local/share/squeekboard/keyboards/custom.yaml) 

---
outlines:
    default: { width: 100, height: 100 }
    action: { width: 200, height: 100 }

views:
    base:
        - "L d1 R d2 F d1 preferences"
        - "d1 ↑ d1 0 1 d1 X d1"
        - "← d1 → d2 Y d1 A"
        - "d1 ↓ d1 N M d1 B d1"

buttons:
    d1:
        action:
            set_view: "base"
        outline: "default"
        label: ""
    d2:
        action:
            set_view: "base"
        outline: "action"
        label: ""
    preferences:
        action: "show_prefs"
        outline: "default"
        icon: "keyboard-mode-symbolic"
    "↑":
        outline: "default"
        keysym: "Up"
    "↓":
        outline: "default"
        keysym: "Down"
    "←":
        outline: "default"
        keysym: "Left"
    "→":
        outline: "default"
        keysym: "Right"

司徒最後發現，官方的Keyboard Layout佈局還是比較適合觸控使用，司徒將原本的Ctrl、Alt改換成Tab、Ctrl(~/.local/share/squeekboard/keyboards/terminal/us.yaml)
