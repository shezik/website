Google Pixel 3a XL >> Droidian

安裝系統

參考資訊：
1. linux-pixel-3a

步驟如下：
1. 刷Android 9到A、B分區 

$ cd
$ wget https://github.com/steward-fu/pixel3axl/releases/download/v1.0/platform-tools_r33.0.3-linux.zip
$ unzip platform-tools_r33.0.3-linux.zip
$ mv platform-tools /tmp
$ export PATH=/tmp/platform-tools:$PATH

$ wget https://github.com/steward-fu/pixel3axl/releases/download/v1.0/bonito-pq3b.190801.002-factory-61a2836d.zip
$ unzip bonito-pq3b.190801.002-factory-61a2836d.zip
$ cd bonito-pq3b.190801.002
$ vim flash-all.sh
    #if ! grep -q partition-exists $(which fastboot); then
    #  echo "fastboot too old; please download the latest version at https://developer.android.com/studio/releases/platform-tools.html"
    #  exit 1
    #fi

$ fastboot -ab
$ fastboot reboot bootloader
$ ./flash-all.sh

$ fastboot -aa
$ fastboot reboot bootloader
$ ./flash-all.sh

P.S. 刷完flash-all.sh後，記得要手動進入fastboot模式

2. 刷Droidian到A分區 

$ cd
$ wget https://github.com/steward-fu/pixel3axl/releases/download/v1.0/droidian-rootfs-api28gsi-arm64_20211127.zip
$ wget https://github.com/steward-fu/pixel3axl/releases/download/v1.0/droidian-adaptation-google-sargo-arm64_20211127.zip
$ wget https://github.com/steward-fu/pixel3axl/releases/download/v1.0/twrp-3.4.0-0-sargo.img

$ fastboot -aa
$ fastboot reboot bootloader
$ fastboot format:ext4 userdata
$ fastboot boot twrp-3.4.0-0-sargo.img

$ adb sideload droidian-rootfs-api28gsi-arm64_20211127.zip
$ adb sideload droidian-adaptation-google-sargo-arm64_20211127.zip

P.S. 要執行sideload前，記得先在TWRP執行sideload選項，然後USB線記得拔插

Droidian的Terminal比Ubunt Touch好用，加上Droidian可以真正使用到48GB空間，比起Ubunt Touch更適合開發者使用，不過，Droidian系統相對不穩定，因此，只建議深度Terminal開發者使用
