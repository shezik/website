手機 - F(x)tec Pro1 - Ubuntu - 安裝系統


參考資料：
https://devices.ubuntu-touch.io/device/pro1/

步驟如下：
1. 按住POWER和VOL-進入Fastboot模式
2. 解鎖Bootloader 

$ sudo fastboot flashing unlock



3. 下載Ubuntu deb(64)並且安裝
4. 執行ubports-installer


5. Select device manually => F(x)tec Pro1/Pro1 X


6. Install


7. Next


8. Continue(Fastboot模式)


9. Continue(Recovery模式)


10. 安裝中




完成
