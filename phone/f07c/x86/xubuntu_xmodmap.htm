定義鍵盤符號(Xubuntu)

由於F-07C的實體鍵盤缺少許多符號鍵，在Linux系統下，操作總是綁手綁腳，因此，司徒只好自己手動定義尚未定義的鍵盤符號，在Linux系統下，有許多方式可以使用，目前司徒使用xmodmap方式，解說如下： 

$ vim ~/.Xmodmap
  keycode 64 = Mode_switch
  keycode 26 = e E braceleft
  keycode 27 = r R braceright apostrophe
  keycode 40 = d D bracketleft
  keycode 41 = f F bracketright
  keycode 44 = j J grave
  keycode 45 = k K bar
  keycode 59 = comma less backslash

設定成開機後自動啟動 

$ vim ~/.xinitrc
  xmodmap ~/.Xmodmap

定義後的鍵盤符號如下

按鍵 符號
Alt + e {
Alt + r }
Alt + d [
Alt + f ]
Shift + , <
Shift + . >
Alt + j `
Alt + k |
Alt + , \
