手機 - PINE64 PinePhone - Mobian - Build Kernel 6.1.12


參考資料：
https://wiki.debian.org/Teams/Mobian/KernelBuild

步驟如下：

$ sudo apt-get install gcc-aarch64* git-buildpackage

$ cd
$ git clone https://salsa.debian.org/Mobian-team/devices/kernels/sunxi64-linux.git pinephone-linux
$ cd cd pinephone-linux/
$ gbp pq import
$ ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- make sunxi64_defconfig
$ ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- make -j4 bindeb-pkg
$ ls -al ../linux-image-6.1.12-00171-gcf1b69217f8a_6.1.12-00171-gcf1b69217f8a-1_arm64.deb
    -rw-r--r-- 1 steward steward 33816824 Feb 24 00:12 ../linux-image-6.1.12-00171-gcf1b69217f8a_6.1.12-00171-gcf1b69217f8a-1_arm64.deb

P.S. 進入mobian系統後，安裝linux-image-6.1.12-00171-gcf1b69217f8a_6.1.12-00171-gcf1b69217f8a-1_arm64.deb，系統會自動更新/boot/extlinux/extlinux.conf的開機選單，另外，arch/arm64/boot/Image.gz就是vmlinuz檔案
