手機 - PINE64 PinePhone - Sailfish OS - 設定鍵盤


參考資料：
https://github.com/sailfish-on-dontbeevil/documentation/wiki/NewUserTips#keyboard-users

步驟如下：

$ cd
$ dconf write /desktop/sailfish/silica/default_allowed_orientations 3

$ devel-su
# zypper in git
# git clone https://codeberg.org/phalio/ppkb-layouts.git
# cd ppkb-layouts
# sh install-xkb.sh
# exit

$ dconf write /desktop/lipstick-jolla-home/layout "'pp'"
$ dconf write /desktop/lipstick-jolla-home/variant "'altgr'"
$ systemctl --user restart lipstick
