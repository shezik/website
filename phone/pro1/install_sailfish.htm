F(x)tec Pro1 >> Sailfish

安裝系統

參考資料：
1. sailfishos-porters-ci
2. 2467-community-build33016updated-18th-aug-sailfish-os-for-fxtec-pro1

步驟如下：
1. 進入Fastboot模式並且解鎖 

$ sudo fastboot flashing unlock

2. 下載Sailfish
3. 命令如下： 

$ cd
$ wget https://github.com/steward-fu/archives/releases/download/pro1/platform-tools_r33.0.3-linux.zip
$ unzip platform-tools_r33.0.3-linux.zip
$ mv platform-tools /tmp
$ export PATH=/tmp/platform-tools:$PATH

$ unzip artifacts.zip
$ unzip sfe-t5-4.4.0.64-devel-20220724/Sailfish_OS--devel-20220724-4.4.0.64-t5-0.0.1.50.zip
$ cd Sailfish_OS--devel-20220724-4.4.0.64-t5-0.0.1.50
$ ./flash.sh

完成
