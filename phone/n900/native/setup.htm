Nokia N900 >> Native Debian

安裝系統

Debian x64(檔案) 

$ xz -d jessie.dd.xz
$ sudo dd if=jessie.dd of=/dev/sdx bs=1M

Maemo 

$ sudo vim /etc/boot.d/30-jessie.item
  ITEM_NAME="debian jessie MicroSD"
  ITEM_KERNEL="uImage"
  ITEM_INITRD="uInitrd"
  ITEM_DEVICE="${EXT_CARD}p1"
  ITEM_FSTYPE="vfat"
  ITEM_OMAPATAG="1"
  ITEM_CMDLINE="root=/dev/mmcblk0p2 rootwait vram=12M console=tty0 quiet"

$ u-boot-update-bootmenu
$ cd /media/mmc1
$ vim run.sh
  wpa_passphrase SSID PASSWD > /etc/wpa_supplicant/wifi.conf
  wpa_supplicant -BDwext -iwlan0 -c/etc/wpa_supplicant/wifi.conf
  dhclient wlan0
$ sudo reboot

P.S. SSID和PASSWD是要連接的Wifi熱點。
