如何在Pidgin上使用Skype聊天軟體

雖然司徒的N900已經可以透過Pidgin執行LINE聊天插件，不過司徒除了使用LINE聊天軟體之外，也會使用Skype聊天軟體，因此，司徒持續搜尋是否有機會在Pidgin上也跑Skype聊天插件，幸運地，司徒找到Pidgin的Skype插件程式碼，下載編譯後，可以在Maemo和Easy Debian系統上執行，只是在Maemo系統下，當Skype進行連線時，會有SSL Handshake的問題，這部份司徒還沒時間解決，不過目前至少Easy Debian系統下，可以使用Pidgin的Skype聊天插件，司徒還是分享一下編譯過程。

安裝套件： 

$ sudo apt-get update
$ sudo apt-get install pidgin libpurple0 libpurple-dev libpurple-bin libboost-dev libboost-test-dev libboost-program-options-dev libevent-dev automake libtool flex bison pkg-config g++ libssl-dev

編譯： 

$ git clone https://github.com/EionRobb/skype4pidgin/tree/master/skypeweb 
$ cd skypeweb
$ make
$ sudo make install
$ sudo cp libskypeweb.so /usr/lib/purple-2/
