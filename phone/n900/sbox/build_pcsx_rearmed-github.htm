build pcsx-rearmed(github)

PCSX-ReARMed是一款相當優秀的PS1模擬器，本身被移植到相當多的平台，其中包含：GP2X Wiz、OpenPandora等，當然Maemo系統也在這份移植名單中，雖然Platform選項支援Maemo系統，不過，司徒測試後，發現，編譯後的顯示效果不好，因為畫面方塊太嚴重，不過將來有時間的話，司徒一定要好好整理、強化PCSX-ReARMed模擬器，目前司徒就先將編譯歩驟整理如下： 

$ sudo apt-get install zlib1g-dev libpng12-dev libsdl1.2-dev pkg-config libhildonfm2-dev g++
$ git clone https://github.com/notaz/pcsx_rearmed.git
$ cd pcsx_rearmed/frontend
$ git submodule update --init --recursive
$ cd ../
$ ./configure --platform=maemo

移除如下兩個tslib項目 

$ vim Makefile
187行 frontend/pl_gun_ts.o
191行 -DHAVE_TSLIB

接著使用make指令進行編譯即可。
