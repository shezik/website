Nokia N900 >> Sbox

build pcsx rearmed 22

編譯步驟如下： 

$ cd 
$ wget https://github.com/steward-fu/n900/releases/download/v1.1/pcsx_rearmed-22.7z
$ 7zr x pcsx_rearmed-22.7z
$ cd pcsx_rearmed-22
$ vim configure
  46: have_armv7="yes"
  47: have_arm_neon="yes"
  48: have_tslib="no"

$ ./configure --platform=maemo --disable-neon
$ vim Makefile
  005: CFLAGS += -Wall -ggdb -Iinclude -ffast-math -mcpu=cortex-a8 -mtune=cortex-a8 -mfloat-abi=softfp
  187: OBJS += maemo/hildon.o maemo/main.o maemo/maemo_xkb.o
  191: CFLAGS += $(shell pkg-config --cflags hildon-1)

$ make

P.S. 遊戲請放在內部Flash(如：MyDocs)，否則很容易發生遊戲Lag的情況。
