Command Line方式一(最高速度為805MHz)

$ sudo chmod 0777 /sys/devices/system/cpu/cpu0/cpufreq/*
$ 500000 > /sys/devices/system/cpu/cpu0/cpufreq/scaling_min_freq 
$ 805000 > /sys/devices/system/cpu/cpu0/cpufreq/scaling_max_freq

Command Line方式二(最高速度為805MHz)

$ kernel-config load lv
$ kernel-config limits 250 805

$ cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_min_freq
$ cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_max_freq

$ kernel-config save oc
$ kernel-config default oc
