自訂Vimrc

這個網站的內容，多數都是在N900上完成，對司徒來說，N900真可稱得上是一部神機，不僅可以當手機使用，更可以當做文書處理和程式開發的機器使用，而司徒主要使用的編輯工具是vim，至於html語法檢查的部份，司徒選擇使用tidy，這兩個工具都可以在maemo系統中安裝使用，不過預設的vim並不適合編輯html檔案，因此，司徒使用如下設定： 

$ vim ~/.vimrc
    set nu
    set nocp
    set ruler
    set paste
    set showcmd
    set bg=light
    set hlsearch
    set showmode
    set expandtab
    set tabstop=4
    set autoindent
    set shiftwidth=4
    set backspace=indent,eol,start
    set enc=utf8
    set fencs=utf8,gbk,gb2312,gb18030
    set fileencodings=utf-8,gb2312,gb18030,gbk,ucs-bom,cp936,latin1
    syntax on

    :map 0 :s!&amp;!\&!ge\|:s!&gt;!>!ge\|:s!&lt;!<!ge<CR>
    :map 1 :s!&!\&amp;!ge\|:s!>!\&gt;!ge\|:s!<!\&lt;!ge<CR>

設定html檔案都使用xml語法 

$ mkdir -p ~/.vim/syntax
$ echo "set syntax=xml" > ~/.vim/syntax/html.vim
vim指令 說明
:e xxx 開啟另一檔案
:e# 關閉開啟的檔案
