安裝Tmux

雖然N900的terminal可以同時開啟多個視窗，不過使用者如果想在同一個terminal裡面再切割成更多的子視窗的話，可以使用tmux套件來完成，安裝過程說明如下介紹。

安裝必要套件： 

$ sudo apt-get install tmux
$ tmux

快捷鍵

按鍵功能
Ctrl + b 再按 c新子視窗
Ctrl + b 再按 "新水平子視窗
Ctrl + b 再按 %新垂直子視窗
Ctrl + b 再按 num切換子視窗
Ctrl + b 再按 o切換水平／垂直子視窗
Ctrl + b 再按 {互換水平／垂直子視窗
Ctrl + b 再按 [進入捲動摸式(按q離開)
Ctrl + space開始選擇
Ctrl + w結束選擇
Ctrl + b 再按 ]貼上
