如何在Pidgin上使用LINE聊天軟體

自從Android手機開始流行後，周遭朋友陸續使用LINE聊天軟體，司徒卻還是只能使用PC版的LINE去跟朋友聊天，因此，司徒曾幾度改拿Android、iPhone和Windows Phone手機，不過最終都還是換回N900，主要的原因在於，司徒還是習慣使用Linux系統，因此，司徒一直在找尋可以在Linux系統上執行LINE聊天軟體的方式，而最近終於在網路上看到有人於Pidgin聊天軟體上，成功執行LINE聊天軟體，而且還是使用Github方式開放源始碼，這真是一個好消息，因為N900也有Pidgin套件，就缺少這個插件，於是司徒花了一些時間測試，最後終於成功在N900的Maemo、Easy Debian系統上，透過Pidgin執行LINE聊天軟體，這真是一個好消息，意謂著司徒的N900還可以再撐個幾年，因此，司徒整理了這篇教學文章，分享給需要LINE聊天軟體的N900使用者。

切換到ScratchBox環境後，安裝如下必要套件： 

$ sudo apt-get update
$ sudo apt-get install pidgin libpurple0 libpurple-dev libpurple-bin libboost-dev libboost-test-dev libboost-program-options-dev libevent-dev automake libtool flex bison pkg-config g++ libssl-dev

編譯歩驟如下： 

$ git clone http://altrepo.eu/git/purple-line.git line
$ cd line/purple-line
$ make thrift
$ make

編譯完成後，只要把libline.so複製到Maemo系統(已安裝Pidgin套件)的/usr/lib/purple-2/就可以，不過由於Maemo系統的libstdc++.so.6太過老舊，因此，司徒還把Easy Debian裡的libstdc++.so.6.0.22複製到Maemo系統使用，Maemo系統的libstdc++.so.6是symlink，因此，只要指到新的libstdc++.so.6.0.22即可。

完成

接著添加完LINE帳號後，Pidgin會跳出LINE的4碼認證號碼，只要在手機上確認後，就可以在N900上使用LINE聊天軟體。
