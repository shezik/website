Nokia N900 >> Maemon Leste

安裝系統

參考資訊：
1. setup
2. images

SDCard 

$ cd
$ wget http://maedevu.maemo.org/images/n900/20191020/maemo-leste-1.0-armhf-n900-20191020.img.xz
$ xz -d maemo-leste-1.0-armhf-n900-20191020.img.xz
$ sudo dd if=maemo-leste-1.0-armhf-n900-20191020.img of=/dev/sdX bs=1M

N900開機選單啟動 

$ sudo vim /etc/bootmenu.d/30-maemo-leste.item
  ITEM_NAME="Maemo Leste"
  ITEM_KERNEL="uImage"
  ITEM_DEVICE="${EXT_CARD}p1"
  ITEM_FSTYPE="ext2"

$ u-boot-update-bootmenu

PC端啟動 

$ wget http://maedevu.maemo.org/images/n900/tools/0xFFFF
$ wget http://maedevu.maemo.org/images/n900/tools/u-boot-2013.04-2.bin
$ chmod a+x
$ sudo ./0xFFFF -m u-boot-2013.04-2.bin -l
$ sudo ./0xFFFF -b
