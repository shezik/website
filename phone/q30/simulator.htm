如何安裝Simulator

Debian 8 x64環境
1. 下載Simulator檔案
2. 執行如下指令 

$ sudo apt-get dist-upgrade
$ sudo apt-get install ia32-libs-multiarch
$ sudo apt-get install ia32-libs
$ chmod a+x BlackBerry10Simulator-Installer-BB10_3_1-995-Linux-201410280147.bin
$ ./BlackBerry10Simulator-Installer-BB10_3_1-995-Linux-201410280147.bin






接著安裝VMware並且加入剛剛安裝的Simulator後，就可以開啟Blackberry 10
