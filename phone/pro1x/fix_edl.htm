F(x)tec Pro1 X

解決開機一直處於edl模式的問題

參考資料：
1. edl
2. pro1-x-backup-restore-and-repair-device-specific-partitions/

步驟如下：
1. 手機關機
2. 同時按住Vol-和Vol+
3. 連接USB到電腦
4. 執行如下命令 

$ sudo apt install adb fastboot python3-dev python3-pip liblzma-dev git
$ sudo apt purge modemmanager

$ cd
$ git clone https://github.com/bkerler/edl
$ cd edl
$ git submodule update --init --recursive
$ sudo cp Drivers/51-edl.rules /etc/udev/rules.d
$ sudo cp Drivers/50-android.rules /etc/udev/rules.d
$ python setup.py build
$ sudo python setup.py install

$ ./edl w persist ./persist_backup.img --memory=ufs
$ sleep 3
$ ./edl w modemst2 ./modemst2_backup.img --memory=ufs
$ sleep 3
$ ./edl w modemst1 ./modemst1_backup.img --memory=ufs
$ sleep 3
$ ./edl w fsc ./fsc_backup.img --memory=ufs
$ sleep 3
$ ./edl w fsg ./fsg_backup.img --memory=ufs
$ sleep 3
