參考資訊：

2. gbdk

編譯gbdk 

$ sudo apt-get install subversion doxygen -y

$ cd
$ svn checkout svn://svn.code.sf.net/p/sdcc/code/trunk/sdcc sdcc
$ cd sdcc
$ ./configure
$ make
$ sudo make install

$ export SDCCDIR=/usr/local

$ cd
$ git clone https://github.com/Zal0/gbdk-2020
$ cd gbdk-2020
$ make
$ make install
$ ls /opt/gbdk/bin/
  bankpack  gbcompress  ihxcheck  lcc  makebin  packihx  png2mtspr  sdar  sdasgb  sdcc  sdcdb  sdcpp  sdldgb  sdnm  sdobjcopy  sdranlib  sz80

編譯gambatte 

$ cd
$ git clone https://github.com/sinamas/gambatte
$ cd gambatte
$ ./build_sdl.sh
$ sudo cp gambatte_sdl/gambatte_sdl /usr/bin/
