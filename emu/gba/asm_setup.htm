編譯器： 

$ cd 
$ wget https://wii.leseratte10.de/devkitPro/devkitARM/r55%20\(2020-07-25\)/devkitARM-r55-1-linux_x86_64.pkg.tar.xz
$ tar xvf devkitARM-r55-1-linux_x86_64.pkg.tar.xz
$ sudo mv opt/devkitpro/devkitARM /opt/
$ /opt/devkitARM/bin/arm-none-eabi-gcc -v
  gcc version 10.2.0 (devkitARM release 55)

$ sudo apt-get install mgba-sdl
