1. nds-docker

司徒目前使用docker當作開發環境，模擬器則是使用mgba，安裝步驟如下： 

$ sudo apt-get install docker* container* mgba-sdl -y

$ cd
$ git clone https://github.com/morgenman/nds-docker
$ cd nds-docker
$ sudo docker build -t nds-env .
