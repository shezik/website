RetroArch

Build All

Ref. http://emulation.gametechwiki.com/index.php/Building_RetroArch

Fetch 

$ cd ~
$ git clone https://github.com/libretro/libretro-super.git
$ cd libretro-super
$ ./libretro-fetch.sh

P.S. 假如只要抓mame下來編譯，就用./libretro-fetch.sh mame

Build RetroArch 

$ cd ~/libretro-super/retroarch
$ git pull
$ git submodule update --init
$ ./configure
$ make clean
$ make -j4

Build Cores 

$ cd ~/libretro-super
$ ./libretro-build.sh

P.S. 假如只要編譯單一個Core，就用./libretro-build.sh xxx

安裝RetroArch 

$ cd ~/libretro-super/retroarch
$ mkdir -p ~/.config/retroarch/assets/
$ cp -a media/assets/* ~/.config/retroarch/assets/
$ sudo make install

安裝Cores 

$ cd ~/libretro-super/
$ mkdir -p ~/.config/retroarch/cores
$ find . -name "*.so" -exec cp {} ~/.config/retroarch/cores
