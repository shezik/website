驅動程式 - Windows Driver Model (WDM) - 解決The third-party INF does not contain digital signature information問題


解法如下： 

[Version]
Signature   = "$Windows NT$"
Provider    = %ProviderName%
Class       = HIDClass
CLASSGUID   = {745a17a0-74d3-11d0-b6fe-00a0c90f57da};
DriverVer   = 05/01/2015,1.0.0.0                    
CatalogFile = Test.cat
LayoutFile  = layout.inf

P.S. 增加一個CatalogFile敘述即可
