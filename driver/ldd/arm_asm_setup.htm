Linux環境對初學者來說，比較棘手的還是系統指令的操作，這部份，使用者只能多多參考其它網站說明，司徒只會將重點放在Linux驅動程式的撰寫上，而司徒目前使用的環境是Debian x64，使用者也可以使用更友好的Ubuntu系統，環境安裝好後，接著製作MicroSD開機系統 

$ cd ~/Downloads
$ wget https://github.com/steward-fu/pocketbeagle/releases/download/v1.0/sdcard.img
$ sudo dd if=sdcard.img of=/dev/sdx bs=1M

接著將此MicroSD插入PocketBeagle，連結PL2303到電腦並且開啟minicom(115200 bps)，就可以看到如下等待進入系統的畫面 

Welcome to PocketBeagle
pocketbeagle login:

P.S. 輸入root就可以進入系統

接著為了避免發生驅動無法掛載的問題，需要自己編譯一次Kernel 

$ cd ~/Downloads
$ wget https://github.com/steward-fu/pocketbeagle/releases/download/v1.0/gcc-4.9.tar.gz
$ tar xvf gcc-4.9.tar.gz
$ sudo mv gcc-4.9 /opt
$ export PATH=/opt/gcc-4.9/bin:$PATH
$ arm-linux-gnueabihf-gcc -v
  gcc version 4.9.4 (Linaro GCC 4.9-2017.01)

$ wget https://github.com/steward-fu/pocketbeagle/releases/download/v1.0/kernel-4.14.108.tar.gz
$ tar xvf kernel-4.14.108.tar.gz
$ cd kernel
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- make pocketbeagle_defconfig
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- make zImage dtbs -j4

P.S. 將編譯後的arch/arm/boot/zImage、arch/arm/boot/dts/am335x-pocketbeagle.dtb覆蓋到MicroSD第一分區的檔案
