開發環境

參考資訊：
1. cubie
2. arduino-cli

如下： 

$ cd
$ wget https://downloads.arduino.cc/arduino-cli/arduino-cli_latest_Linux_64bit.tar.gz
$ tar xvf arduino-cli_latest_Linux_64bit.tar.gz
$ sudo mv arduino-cli /usr/local/bin/
$ arduino-cli core install arduino:avr
