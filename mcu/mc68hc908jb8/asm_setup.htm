開發環境

參考資訊：
1. asmlnk
2. hcs08rmv1
3. funwith08
4. mc68hc908jb8
5. blink-with-sdas

司徒目前使用SDCC編譯器，而燒錄軟體則是使用bl08，安裝方式如下： 

$ sudo apt-get install sdcc -y

$ cd
$ wget https://github.com/steward-fu/mc68hc908jb8/releases/download/v1.0/bl08.c
$ gcc bl08.c -o bl08
$ sudo cp bl08 /usr/local/bin
$ sudo chmod 0777 /usr/local/bin/bl08
$ bl08
