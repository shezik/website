編譯： 

$ cd
$ git clone https://github.com/steward-fu/a33_all
$ cd a33_all/linux-3.4
$ cp a33_vstar_defconfig .config
$ cd ../buildroot/target/dragonboard
$ 7za x rootfs.tar.gz.zip
$ 7za x sysroot.tar.gz.zip
$ cd ../../dl/
$ 7za x gcc-linaro.tar.bz2.zip
$ cd ../../
$ ./build.sh config
