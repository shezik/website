ATtiny13

解決"avrdude: error: program enable: target doesn't answer"問題

參考資訊：
1. how-to-update-avr-usbasp-firmware-to-latest-version

加上-B 1024設定bitclock，如下命令測試： 

$ avrdude -p attiny13 -c usbasp -B 1024 -F -P usb
  avrdude: set SCK frequency to 500 Hz
  avrdude: AVR device initialized and ready to accept instructions
  Reading | ################################################## | 100% 0.20s
  avrdude: Device signature = 0x1e9007 (probably t13)
  avrdude: safemode: Fuses OK (E:FF, H:FF, L:6A)
