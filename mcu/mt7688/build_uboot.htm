LinkIt Smart MT7688

Build UBoot

參考資訊：
1. build-the-bootloader

編譯： 

$ cd
$ wget https://github.com/steward-fu/mt7688/releases/download/v1.0/buildroot-gcc342.tar.bz2
$ tar xvf buildroot-gcc342.tar.bz2
$ sudo mv buildroot-gcc342 /opt
$ export PATH=$PATH:/opt/buildroot-gcc342/bin

$ wget https://github.com/steward-fu/mt7688/releases/download/v1.0/uboot.tar.gz
$ tar xvf uboot.tar.gz
$ cd uboot
$ make

燒錄(請使用USB供電)：
1. 格式化USBDisk成FAT32
2. 複製uboot.bin到USBDisk(lks7688.ldr)
3. 插入USBDisk到MT7688 USB host port
4. 按住WiFi按鈕
5. 20秒後鬆開WiFi按鈕(第二次WiFi LED亮時)
