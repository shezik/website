(ARM Cortex-A7) T113-S3 (MangoPi MQ-R)

如何從MicroSD啟動

參考資料：
1. awboot
2. whycan
3. debugdump

MicroSD：


步驟如下： 

$ cd
$ git clone https://github.com/szemzoa/awboot
$ cd awboot
$ make
$ sudo dd if=awboot-boot-sd.bin of=/dev/sdX bs=1024 seek=8

$ cd
$ git clone https://github.com/Evlers/linux_kernel_t113
$ cd linux_kernel_t113
$ make mangopi_dual_t113_defconfig
$ make zImage dtbs modules -j4
$ cp arch/arm/boot/zImage /xxx
$ cp arch/arm/boot/dts/sun8i-t113-mangopi-dual.dtb /xxx
