如何透過teensy_loader_cli燒錄程式

參考資訊：
1. bareteensy31
2. teensy_loader_cli
3. bare-metal-assembly-on-the-teensy-3.1

步驟如下：
1. 按住按鍵不放
2. 連接USB與PC
3. 放開按鍵
4. 偵測到的裝置 

$ sudo dmesg 
  [372357.762690] usb 3-1: new full-speed USB device number 99 using xhci_hcd
  [372357.903329] usb 3-1: New USB device found, idVendor=16c0, idProduct=0478
  [372357.903344] usb 3-1: New USB device strings: Mfr=0, Product=0, SerialNumber=1
  [372357.903346] usb 3-1: SerialNumber: 0008B76F
  [372357.904410] hid-generic 0003:16C0:0478.000C: hidraw2: USB HID v1.11 Device [HID 16c0:0478] on usb-0000:00:14.0-1/input0

5. 執行如下命令 

$ sudo teensy_loader_cli --mcu=mk20dx256 -w xxx.hex
