掌機 - (ARM Cortex-A53) S32G399 - Build Kernel


步驟如下： 

$ cd
$ git clone https://github.com/nxp-auto-linux/linux --recursive
$ cd linux
$ ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- make s32cc_defconfig
$ ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- make all
$ INSTALL_MOD_PATH=out ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- make modules_install
