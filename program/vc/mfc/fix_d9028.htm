解決"warning D9028: minimal rebuild failure"問題

若在Visual Studio 2008出現Command line warning D9028: minimal rebuild failure問題時，代表mspdb80.dll檔案有問題，刪除多餘的mspdb80.dll檔案即可，常見的存在位置如下 

C:/Program Files/Microsoft Visual Studio 9.0/Common7/IDE
C:/Program Files/Microsoft Visual Studio 9.0/Team Tools/Performance Tools
C:/Program Files/Microsoft Visual Studio 9.0/VC/bin
C:/Program Files/Microsoft Visual Studio 9.0/VC/bin/ia64
C:/Program Files/Microsoft Visual Studio 9.0/VC/bin/amd64
C:/Program Files/Common Files/Microsoft Shared/VSA/9.0/VsaEnv

P.S. 若在Visual Studio 2010出現此錯誤，只要刪除*.idb檔案再重新編譯專案即可
