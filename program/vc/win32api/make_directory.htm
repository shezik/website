MakeSureDirectoryPathExists

CreateDirectory函式是使用者常用來產生資料夾的函式，但是該函式無法產生多層不存在的資料夾，使用者若要產生多層不存在的資料夾就必須自己判斷\\字串，一層一層透過CreateDirectory函式來產生，其實有一個更好用的函式，它就是MakeSureDirectoryPathExists函式，用法跟CreateDirectory函式一樣簡單，但是，MakeSureDirectoryPathExists函式可以產生多層不存在的資料夾，因此，使用者再也不用一層一層建立

該函式宣告如下 

BOOL WINAPI MakeSureDirectoryPathExists(_In_ PCSTR DirPath);
