使用C語言刷題時，可以使用qsort()做排序，qsort()預設使用QuickSort演算法，時間複雜度是n*log(n) 

int values[] = {40, 10, 100, 90, 20, 25};

int compare(const void *a, const void *b)
{
    return (*(int*)a - *(int*)b);
}

int main(int argc, char **argv)
{
    qsort(values, 6, sizeof(int), compare);
    return 0;
}

values 

10 20 25 40 90 100
