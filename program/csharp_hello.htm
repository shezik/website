程式語言 - C# - Hello, world!


參考資訊：
https://www.mono-project.com/
https://www.geeksforgeeks.org/how-to-compile-decompile-and-run-c-code-in-linux/

main.cs 

using System;

public class main {
    static public void Main() {
        Console.WriteLine("Hello, world!");
    }
}

編譯、執行 

$ mcs -out:main.exe main.cs
$ mono main.exe
    Hello, world!
