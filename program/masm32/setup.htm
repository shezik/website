開發環境

司徒很早就接觸MASM32程式，也相當驚訝國外竟然有論壇在維護MASM32 SDK開發工具，司徒記得當初還購買過MASM32的介紹書籍，一窺MASM32的魅力。程式開發工具，如果具備短小精幹的可攜性，應該是許多工程師的最愛，而剛好MASM32具備了這個特點，這也是司徒相當喜歡MASM32的原因，爲此，司徒總是希望自己的教學網站可以有一系列的教學文章，藉此推廣使用MASM32開發程式，只可惜司徒也是一個相當懶惰的人，因此，就這樣一直耽擱著，直到最近，司徒才發現，該是報恩的時候了，因此，這才有了這一系列的MASM32教學文章。

基本上，MASM32 SDK是運行在Windows系統上的開發工具，只是，司徒的電腦都是安裝Debian系統，因此，司徒只好用Wine來運行MASM32，基本上是無差別的，所以使用者不用擔心是否會有不一樣的地方，以下是安裝步驟： 

$ sudo apt-get install wine
$ cd ~/Downloads/
$ wget http://website.assemblercode.com/masm32/masm32v11r.zip
$ unzip masm32v11r.zip
$ wine install.exe
