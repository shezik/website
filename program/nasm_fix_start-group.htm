程式語言 - Netwide Assembler (NASM) - 解決Undefined reference to dl_iterate_phdr問題


問題如下： 

ld: /usr/lib/gcc/i686-linux-gnu/8/libgcc_eh.a(unwind-dw2-fde-dip.o): in function `_Unwind_Find_FDE':
(.text+0x19cc): undefined reference to `dl_iterate_phdr'

解法如下： 

$ ld -melf_i386 main.o -o main -static --start-group libxxx.a --end-group

P.S. 使用--start-group和--end-group圈住libxxx.a
