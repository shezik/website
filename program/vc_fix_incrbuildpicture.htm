程式語言 - Visual C++ 6.0 - 解決Internal error during IncrBuildImage問題


Visual Studio C++ 2008在Win7出現Internal error during IncrBuildImage錯誤的解法：
1. 修改Project => Property => Linker => General => Enable Incremental Linking成"No (/INCREMENT:NO)"
2. 修改Project => Property => C/C++ => General => Debug Information Format成Program Database (/Zi)
