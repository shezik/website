SDL_SWSURFACE、SDL_HWSURFACE

區別如下：

SDL_SWSURFACE從系統記憶體配置使用(system memory)
SDL_HWSURFACE從顯示卡記憶體配置使用(video memory if possible), 讀寫Pixels時，需要先呼叫SDL_LockSurface()鎖住記憶體
