開發環境

參考資訊：
1. fasm

司徒目前是在Pro1 X上撰寫FASM相關的教學程式，而使用的工具則是box86 + wine，比較特別的是，司徒的手機是安裝Sailfish OS，所以司徒是再安裝LXC(Debian 12 ARM64)，讓FASM程式跑在這個Container裡面，安裝box64 + wine的步驟如下： 

$ cd
$ wget https://raw.githubusercontent.com/ThieuMinh26/Proot-Setup/main/Box86-64_Wine86-64.sh
$ chmod a+x ./Box86-64_Wine86-64.sh
$ ./Box86-64_Wine86-64.sh

$ WINEPREFIX=~/.wine_amd64 box86 wine winecfg

DPI設定成144


接着使用司徒已經解開的FASM檔案 

$ cd
$ git clone https://github.com/steward-fu/win32
$ cp -a win32/fasm ~/.wine_amd64/drive_c/

接着測試是否可以正常執行程式 

$ WINEPREFIX=~/.wine_amd64 box86 wine ~/.wine_amd64/drive_c/fasm/fasmw.exe


P.S. 所有需要的Share Library(*.so and *.dll)都必須成功載入，否者將無法正確執行Windows PE程式
