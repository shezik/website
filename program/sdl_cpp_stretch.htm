程式語言 - Simple DirectMedia Layer (SDL) - v1.2 - C/C++ - Stretch Surface


參考資訊：
https://www.libsdl.org/release/SDL-1.2.15/docs/html/index.html

main.c 

#include <stdio.h>
#include <stdlib.h>
#include <SDL.h>

int main(int argc, char **argv)
{
    SDL_Surface *screen = NULL;

    SDL_Init(SDL_INIT_VIDEO);
    screen = SDL_SetVideoMode(320, 240, 16, SDL_HWSURFACE);

    SDL_Surface *bmp = SDL_LoadBMP("main.bmp");
    SDL_SoftStretch(bmp, NULL, screen, NULL);
    SDL_Flip(screen);
    SDL_Delay(3000);

    SDL_FreeSurface(bmp);
    SDL_Quit();
    return 0;
}

P.S. 圖片像素格式必須跟SDL_SetVideoMode()一致，可以使用ffmpeg -i src.jpg -pix_fmt rgb565 -y dst.bmp轉換或者使用SDL_ConvertSurface()轉換

編譯、執行 

$ gcc main.c -o main -lSDL -I/usr/include/SDL
$ ./main
