逆向工程 - Ghidra - Pyhidra - 安裝環境


參考資訊：
https://pypi.org/project/pyhidra/
https://github.com/dod-cyber-crime-center/pyhidra

自動安裝： 

$ pip3 install pyhidra

手動安裝： 

$ sudo apt-get install python3-dev -y

$ cd
$ wget https://files.pythonhosted.org/packages/42/14/b5cb4e4ff97fe33e3f45a0ff8a28ba01cfc60e5878516fda1e08afe8b861/pyhidra-1.0.2.tar.gz
$ tar xvf pyhidra-1.0.2.tar.gz
$ cd pyhidra-1.0.2/
$ sudo python3 setup.py build
$ sudo python3 setup.py install
