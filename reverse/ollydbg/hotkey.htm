快速鍵

如下表：

快速鍵 說明 用途
* 返回目前EIP位置 預覽程式後，可以快速返回當前EIP位置
- 返回上一個位置 預覽進入一個副程式後，可以使該指令返回上一個副程式
+ 進入副程式 在CALL指令處按下該指令，可以進入該副程式預覽
Enter 進入副程式 在CALL指令處按下該指令，可以進入該副程式預覽
Ctrl + * 強制從目前位址開始執行 強制Debug片段程式
Ctrl + n 列出目前Module的Win32 API清單 Win32 API斷點設定
Ctrl + F9 自動執行到RET位置 跳離目前副程式
Alt + F9 自動執行到User Code位置 從Win32 API斷點返回User Code
Ctrl + g 跳到指定的位址 預覧程式使用(也適用於Win32 API斷點)
