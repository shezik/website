繪圖相關 - KiCAD - 0.20140622 BZR4027 - 如何把SVG轉成Footprint元件


參考資訊：
https://github.com/mtl/svg2mod

步驟如下：
1. 安裝svg2mod 

$ cd
$ git clone https://github.com/mtl/svg2mod
$ cd svg2mod
$ sudo python setup.pu install

2. 安裝Inkscape 

$ sudo apt-get install inkscape

3. 製作Footprint圖形
