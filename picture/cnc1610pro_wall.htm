繪圖相關 - CNC1610 Pro - 自製擋板


由於司徒的CNC1610 Pro沒有擋板，深怕PCB屑屑亂飛，只好製作一個擋板


OpenSCAD 

translate([30, 30, 0]) {
	union() {
		translate([0, 6, 0]) {
			cube([6, 100, 70]);
		}
		cube([110, 6, 63]);
	}
}

translate([60, 60, 0]) {
	union() {
		translate([0, 6, -7]) {
			cube([6, 100, 70]);
		}
		cube([110, 6, 63]);
	}
}

difference() {
	cube([60, 10.5, 66]);
	translate([0, 2, 0]) {
		cube([60, 6.5, 64]);
	}
}
