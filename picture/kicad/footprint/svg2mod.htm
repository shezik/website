如何把SVG轉成Footprint元件

[Github] svg2mod

1. 安裝svg2mod 

$ cd
$ git clone https://github.com/mtl/svg2mod
$ cd svg2mod
$ sudo python setup.pu install

2. 安裝Inkscape 

$ sudo apt-get install inkscape

3. 製作footprint圖形
