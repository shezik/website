系統 - Debian - QEMU - 如何儲存、載入Snapshot


參考資訊：
https://serverfault.com/questions/889163/combine-snapshot-and-loadvm-snap-id-in-qemu-system

儲存Snapshot： 

$ qemu-system-i386 -hda xxx.qcow2 -boot c -cpu coreduo -m 1024 -vga cirrus -net nic,model=pcnet -rtc base=localtime -accel tcg -no-hpet -serial none -parallel none -monitor stdio -snapshot

(qemu) migrate "exec: gzip -c > myvm.gz"

載入Snapshot： 

$ qemu-system-i386 -hda xxx.qcow2 -boot c -cpu coreduo -m 1024 -vga cirrus -net nic,model=pcnet -rtc base=localtime -accel tcg -no-hpet -serial none -parallel none -monitor stdio -incoming "exec: gzip -c -d myvm.gz" -snapshot
