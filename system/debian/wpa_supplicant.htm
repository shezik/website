wpa_supplicant

如果使用者需要使用無線網卡上網並且使用WPA/WPA2加密連線時，可以使用如下指令進行連線(iwconfig無法支援WPA/WPA2加密連線)

設定新的無線AP(無線AP="Test" 密碼="12345678") 

$ sudo apt-get install wpasupplicant
$ sudo wpa_passphrase "Test" "12345678" > /etc/wpa_supplicant/wifi.conf

連線(電腦重新開機後，只需使用如下指令連線即可) 

$ sudo wpa_supplicant -BDwext -iwlan0 -c/etc/wpa_supplicant/wifi.conf
$ sudo dhclient wlan0
