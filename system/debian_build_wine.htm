系統 - Debian - Build Wine


參考資訊：
https://github.com/Winetricks/winetricks
https://www.reddit.com/r/wine_gaming/comments/7xgux5/wine_3_how_to_build_a_proper_working_wine_for/

步驟如下： 

$ cd
$ git clone git://source.winehq.org/git/wine.git
$ cd wine
$ mkdir -p build/wine-32
$ mkdir -p build/wine-64

$ cd build/wine-64
$ ../../wine/configure CFLAGS="-march=native -O3 -pipe -fstack-protector-strong" --enable-win64
$ make -j4
$ sudo make install

$ cd ../wine-32
$ PKG_CONFIG_PATH=/usr/lib/pkgconfig ../../configure CFLAGS="-march=native -O3 -pipe -fstack-protector-strong" --with-wine64=../wine-64
$ make -j4
$ sudo make install

$ export WINEARCH=win32
$ export WINEPREFIX=/home/user/.wine
$ wine winecfg
