系統 - Automotive Stream Distribution (AutoSD) - 安裝系統(QEMU aarch64)


參考資訊：
https://sig.centos.org/automotive/
https://autosd.sig.centos.org/AutoSD-9/nightly/sample-images/

安裝步驟： 

$ sudo apt-get install qemu qemu-kvm virt-manager bridge-utils -y
$ sudo useradd -g $USER libvirt
$ sudo useradd -g $USER libvirt-kvm
$ sudo systemctl enable libvirtd.service
$ sudo systemctl start libvirtd.service

$ cd
$ wget https://autosd.sig.centos.org/AutoSD-9/nightly/sample-images/auto-osbuild-qemu-autosd9-developer-regular-aarch64-1245137814.94376611.qcow2.xz
$ xz -d auto-osbuild-qemu-autosd9-developer-regular-aarch64-1245137814.94376611.qcow2.xz

$ virt-manager



File => New Virtual Machine


選擇auto-osbuild-qemu-autosd9-developer-regular-aarch64-1245137814.94376611.qcow2


Memory=1024, CPU=1


Firmware要記得選


選擇Begin Installation後，就會開始啟動AutoSD系統，預設密碼(root:password)
