系統 - Debian - QEMU - 安裝Windows 98


參考資訊：
https://archive.org/details/windows-98-se-isofile
https://www.lysator.liu.se/~jonass/qemu_win98.html

步驟如下： 

$ qemu-img create -f qcow2 win98.img 1G
$ qemu-img create -f qcow2 data.qcow2 1G
$ qemu-system-i386 -hda win98.img -drive file=windows98se.iso,if=ide,index=1,media=cdrom -boot c -cpu coreduo -m 512 -vga cirrus -net nic,model=pcnet -net user,smb=/tmp -rtc base=localtime -accel tcg -no-hpet -device AC97 -drive file=data.qcow2 -serial none -parallel none

P.S. 安裝完成後再掛載data.qcow2

掛載data.qcow2 

$ sudo modprobe nbd max_part=8
$ sudo qemu-nbd --connect=/dev/nbd0 ./data.qcow2 
$ sudo fdisk /dev/nbd0 -l
$ sudo mount /dev/nbd0p1 /mnt

卸載data.qcow2 

$ sudo umount /mnt
$ sudo qemu-nbd --disconnect /dev/nbd0
$ sudo rmmod nbd
